#Jobnavi-任务管理系统
一个大数据客户端任务的部署、发布、调度系统。

##Jobnavi支持的特性
注册定时作业或一次执行作业

发布多语言任务

任务重运行

HA

运行节点退服与扩容

任务资源动态分配算法

调度作业优先级管理

权限体系管理

客户端任务自定义事件

运行节点黑名单

运行节点支持配置标签

Jobnavi热升级

作业资源隔离

调度作业分组管理

Metric

元数据过期清理

父子任务依赖规则配置

任务延迟启动

...
